Раздел «Телефонные списки»

Телефонные списки — список номеров телефонов, которые нужно обзвонить. Один и тот же телефонный список может быть использован разными кампаниями.
Например, это могут быть ваши клиенты, которым вы хотите что-то сообщить и получить обратную связь.
  • Что такое телефонные списки
  • Допустимые форматы номеров
  • Способы добавления номеров
  • Создание телефонных списков
Мы не предоставляем базы телефонных номеров. Телефонные списки вы формируете самостоятельно.

Особенности работы с телефонными списками

Телефонные списки должны задаваться заранее для кампаний, имеющих тип «Обзвон по списку» и «Конструктор роликов».
Для остальных кампаний телефонные списки изначально не задаются и будут пустыми, с автоматически сгенерированными названиями, которые начинаются с «From campaign …

Телефонные списки с названием в стиле «From campaign…» ожидают, что вы будете добавлять номера на прозвон на лету через API, но схема работы у них одинаковая.
Иногда вам требуется соотнести дополнительную информацию с номером телефона.
Например, вы хотите обратиться к вашему клиенту по имени или продиктовать внутренний номер заказа, для этого вы можете использовать столбцы.
При добавлении номера телефона укажите дополнительную информацию, и вы сможете её использовать при формировании шаблона аудиоролика.

3 способа добавления номеров телефонов

  • Вставка телефонов в поле Простой способ создания телефонного списка. Вставляйте номера по одному в строке.
  • Ручное создание списка номеров Если вы хотите, чтобы были заполнены данные «Столбец №» добавьте значения через запятую.
  • Создание списка из вашего файла Нажмите «Выберите файл» и загрузите свой список. Файл должен быть в формате XLS/CSV.
Вводите номера в любом удобном формате. Если у вас есть список номеров — просто вставьте его в поле и список готов.
Каждый номер обязательно должен быть в отдельной строке!
Например, строка: «89000000000, Анастасия, 555» создаст телефонный список, где «Столбец 1» будет содержать значение Анастасия, «Столбец 2» — 555.
Столбец с номерами считается нулевым!
Если вы хотите использовать столбцы, следующая колонка после номера телефона будет содержать значение для «Столбец 1» и так далее по порядку слева направо.
Номера телефонов должны быть в первом столбце!
При создании телефонного списка вы можете использовать разный формат написания номера.
  • 8 900 000-00-00
  • 900 000 00 00
  • +7 (900) 000-00-00
  • 79000000000

Значение всех полей в разделе «Телефонные списки»

  • 1. Можно добавить телефонный список.
  • 2. В этом поле можно найти список по названию или в каком списке есть определенный номер.
  • 3. Активные телефонные списки — это те, которые используются какой-либо кампанией.
    В архиве хранятся списки, которые устарели и больше не нужны (вы сами добавляете списки в архив).
  • 4. Каждому списку присваивается свой ID.
  • 5. Здесь отображается название ваших списков.
    Стоп-лист есть у каждого пользователя. В него попадают номера абонентов, которые не хотят, чтобы им звонили.
    Если номер есть в стоп-листе, то звонки на него проходить не будут.
  • 6. Дата, когда список был создан.
  • 7. Действия, которые можно совершать со списком.

Создание телефонного списка

Телефонный список добавляется двумя способами:
  • В разделе «Телефонные списки» нажмите кнопку «Добавить».
  • При создании компании нажмите кнопку «Создать новый список».
  • 1. Введите название телефонного списка.
  • 2. Если нужно, загрузите Excel-файл с списком.
  • 3. Либо вручную вводите номера в поле. По одному номеру в строке.
  • 4. После добавления номеров нажмите «Создать».

Стоимость звонка — 1,7 ₽/мин (посекундная тарификация)
При обзвоне система отслеживает более 20 параметров. От количества операторов в линии до среднего времени разговора и доли недозвонов.
На основе анализа система прогнозирует будущую нагрузку и совершает такое количество звонков, чтобы все операторы были заняты, а клиенты не слушали гудки.
Система автоматически соединяет оператора с клиентом без гудков и долгих ожиданий.
Звонок распознаёт автоответчики и не тратит на них деньги и внимание операторов.
На основе анализа данных система рассчитывает, когда пора звонить следующему клиенту, чтобы сразу соединить его с освободившимся оператором.
Вы можете сами настроить паузы между звонками, степень нагрузки и другие параметры.
